6.2.1
Overview
1
This section attempts to exhaustively document all of the possible configuration parameters.
6.2.2
com.io7m.jfprop.fossil_executable
1
The com.io7m.jfprop.fossil_executable parameter specifies the path to the fossil executable that should be used by the server for performing synchronizations.
6.2.3
com.io7m.jfprop.fossil_repository_directory
1
The com.io7m.jfprop.fossil_repository_directory parameter specifies the directory containing Fossil repositories that the server will manage.
6.2.4
com.io7m.jfprop.fossil_user_name
1
The com.io7m.jfprop.fossil_user_name parameter specifies the username that will be used to access Fossil repositories.
6.2.5
com.io7m.jfprop.mail_auth_password
1
The com.io7m.jfprop.mail_auth_password parameter specifies the password that will be used to authenticate to the mail server given by com.io7m.jfprop.mail_server_address.
6.2.6
com.io7m.jfprop.mail_auth_user
1
The com.io7m.jfprop.mail_auth_user parameter specifies the username that will be used to authenticate to the mail server given by com.io7m.jfprop.mail_server_address.
6.2.7
com.io7m.jfprop.mail_enabled
1
The com.io7m.jfprop.mail_enabled parameter specifies whether or not mail notifications are enabled. Must be true or false.
6.2.8
com.io7m.jfprop.mail_recipient
1
The com.io7m.jfprop.mail_recipient parameter specifies the address of the recipient for mail notifications.
6.2.9
com.io7m.jfprop.mail_sender
1
The com.io7m.jfprop.mail_sender parameter specifies the address of the sender for mail notifications; it is the address from which notifications will appear to originate.
6.2.10
com.io7m.jfprop.mail_server_address
1
The com.io7m.jfprop.mail_server_address parameter is the address of the external mail server that will be contacted in order to send mail notifications.
6.2.11
com.io7m.jfprop.server_admin_address
1
The com.io7m.jfprop.server_admin_address parameter is the address to which the administration HTTP server will bind.
6.2.12
com.io7m.jfprop.server_admin_enabled
1
The com.io7m.jfprop.server_admin_enabled parameter specifies whether or not the administration HTTP server is enabled. Must be true or false.
6.2.13
com.io7m.jfprop.server_admin_https_address
1
The com.io7m.jfprop.server_admin_https_address parameter specifies the address to which the administration HTTPS server will bind.
6.2.14
com.io7m.jfprop.server_admin_https_enabled
1
The com.io7m.jfprop.server_admin_https_enabled parameter specifies whether or not the administration HTTPS server is enabled. Must be true or false.
6.2.15
com.io7m.jfprop.server_admin_https_keystore_password
1
The com.io7m.jfprop.server_admin_https_keystore_password parameter specifies the password used to access the administration HTTPS server keystore.
6.2.16
com.io7m.jfprop.server_admin_https_keystore_path
1
The com.io7m.jfprop.server_admin_https_keystore_path parameter specifies the path of the keystore that will be used for the administration HTTPS server.
6.2.17
com.io7m.jfprop.server_admin_https_keystore_type
1
The com.io7m.jfprop.server_admin_https_keystore_type parameter specifies the type of the administration HTTPS server keystore. Will typically be "JKS" as this is the only guaranteed-available keystore type on the Java platform.
6.2.18
com.io7m.jfprop.server_admin_https_port
1
The com.io7m.jfprop.server_admin_https_port parameter specifies the port that the administration HTTPS server will use.
6.2.19
com.io7m.jfprop.server_admin_https_truststore_password
1
The com.io7m.jfprop.server_admin_https_truststore_password parameter specifies the password used to access the administration HTTPS server truststore.
6.2.20
com.io7m.jfprop.server_admin_https_truststore_path
1
The com.io7m.jfprop.server_admin_https_truststore_path parameter specifies the path of the truststore that will be used for the administration HTTPS server.
6.2.21
com.io7m.jfprop.server_admin_https_truststore_type
1
The com.io7m.jfprop.server_admin_https_truststore_type parameter specifies the type of the administration HTTPS server truststore. Will typically be "JKS" as this is the only guaranteed-available truststore type on the Java platform.
6.2.22
com.io7m.jfprop.server_admin_password
1
The com.io7m.jfprop.server_admin_password parameter specifies the password that will be used to access the administration server.
6.2.23
com.io7m.jfprop.server_admin_port
1
The com.io7m.jfprop.server_admin_port parameter specifies the port that the administration HTTP server will use.
6.2.24
com.io7m.jfprop.server_database_file
1
The com.io7m.jfprop.server_database_file parameter specifies the path to the server's database file. Will be created if it does not already exist.
6.2.25
com.io7m.jfprop.server_http_address
1
The com.io7m.jfprop.server_http_address parameter specifies the address to which the HTTP server will bind.
6.2.26
com.io7m.jfprop.server_http_enabled
1
The com.io7m.jfprop.server_http_enabled parameter specifies whether or not the HTTP server is enabled. Must be true or false.
6.2.27
com.io7m.jfprop.server_http_port
1
The com.io7m.jfprop.server_http_port parameter specifies the port that the HTTP server will use.
6.2.28
com.io7m.jfprop.server_https_address
1
The com.io7m.jfprop.server_https_address parameter specifies the address to which the HTTPS server will bind.
6.2.29
com.io7m.jfprop.server_https_enabled
1
The com.io7m.jfprop.server_https_enabled parameter specifies whether or not the HTTPS server is enabled. Must be true or false.
6.2.30
com.io7m.jfprop.server_https_keystore_password
1
The com.io7m.jfprop.server_https_keystore_password parameter specifies the password used to access the HTTPS server keystore.
6.2.31
com.io7m.jfprop.server_https_keystore_path
1
The com.io7m.jfprop.server_https_keystore_path parameter specifies the path of the keystore that will be used for the administration HTTPS server.
6.2.32
com.io7m.jfprop.server_https_keystore_type
1
The com.io7m.jfprop.server_https_keystore_type parameter specifies the type of the HTTPS server keystore. Will typically be "JKS" as this is the only guaranteed-available keystore type on the Java platform.
6.2.33
com.io7m.jfprop.server_https_port
1
The com.io7m.jfprop.server_https_port parameter specifies the port that the HTTPS server will use.
6.2.34
com.io7m.jfprop.server_https_truststore_password
1
The com.io7m.jfprop.server_https_truststore_password parameter specifies the password used to access the HTTPS server truststore.
6.2.35
com.io7m.jfprop.server_https_truststore_path
1
The com.io7m.jfprop.server_https_truststore_path parameter specifies the path of the truststore that will be used for the administration HTTPS server.
6.2.36
com.io7m.jfprop.server_https_truststore_type
1
The com.io7m.jfprop.server_https_truststore_type parameter specifies the type of the HTTPS server truststore. Will typically be "JKS" as this is the only guaranteed-available truststore type on the Java platform.
6.2.37
com.io7m.jfprop.server_log_directory
1
The com.io7m.jfprop.server_log_directory parameter specifies the base log directory for the server. The server will typically create subdirectories inside this directory for storing access logs.
6.2.38
com.io7m.jfprop.server_log_level
1
The com.io7m.jfprop.server_log_level parameter specifies the level of logging to be used. Must be one of LOG_DEBUG, LOG_INFO, LOG_WARN, LOG_ERROR, or LOG_CRITICAL.
6.2.39
com.io7m.jfprop.server_mass_synchronizer_enabled
1
The com.io7m.jfprop.server_mass_synchronizer_enabled parameter specifies whether or not the mass synchronizer is enabled. Must be true or false.
