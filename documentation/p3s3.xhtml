3.3
user_name
3.3.1
Description
1
Values of the user_name type represent valid user names.
3.3.2
Definition
1
A string s can be interpreted as a value of type user_name iff:
3.3.2.1. Type predicate
(empty(s) = false) ∧ ∀i. in(s, i), 
  ∀c. c = get(s,i),
    is_digit(c) ∨ is_letter(c) ∨ (c = U+005F)
3.3.3
Java
1
In the io7m-jfprop server implementation, the user_name type is represented by the JFPUserName type.
