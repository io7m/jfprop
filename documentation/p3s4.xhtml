Values of the key type represent valid keys.
3.4.2
Definition
1
A string s can be interpreted as a value of type key iff:
3.4.2.1. Type predicate
(empty(s) = false) ∧ ∀i. in(s, i), 
  ∀c. c = get(s,i),
    is_digit(c) ∨ (c >= U+0061 ∧ c <= U+0066)
3.4.3

1
In the io7m-jfprop server implementation, the key type is represented by the JFPKey type.
